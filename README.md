# 감쇠 미분방정식과 푸리에 변환을 활용한 포르피린 흡수 스펙트럼의 피크 위치와 선폭 분석

## 탐구 목적
감쇠 상수 γ가 시간 영역의 전자 진동과 주파수 영역의 흡수 스펙트럼에 미치는 영향을 미분방정식과 푸리에 변환으로 분석하는 것이다.

## 사용한 수학 개념
- 미분방정식
- 지수함수
- 삼각함수
- 푸리에 변환
- 함수의 극값
- 반치폭 계산

## 핵심 모델
- \( \frac{dA}{dt} = -\gamma A \)
- \( A = A_0 e^{-\gamma t} \)
- \( P = A_0 e^{-\gamma t} \sin(\omega_0 t) \)
- \( I \propto \frac{1}{\gamma^2 + (\omega - \omega_0)^2} \)
- 반치폭 = \( 2\gamma \)

## 인터랙티브 페이지 설명
감쇠 상수 γ와 중심 주파수 ω₀를 조절하면 시간 영역의 감쇠 진동과 주파수 영역의 흡수 스펙트럼이 함께 변화한다. 이를 통해 γ가 커질수록 진동이 빠르게 사라지고, 흡수 피크의 폭이 넓어짐을 확인할 수 있다.
배포 주소: [https://yuriluv.github.io/calculus](https://yuriluv.github.io/calculus)

## 결과 해석
수기 계산에서 도출한 반치폭 2γ 관계가 그래프와 일치함을 확인하였다. 따라서 감쇠 미분방정식과 푸리에 변환을 이용해 흡수 스펙트럼의 피크 위치와 선폭을 정량적으로 분석할 수 있었다.
